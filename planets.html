Rules

Object of the Game:

The first player to get to the Rebel Alliance base is the winner.

Game Setup:

The game board must be set up in a way that all players can view it.

Turn-Based Gameplay:

The game is turn-based, with each player taking a turn to make a move. The order of play can be determined randomly, by age or a real life lightsaber battle.

Movement:

Players roll the dice, and their game pieces will move according to the rules of the game.

Chance:

If a player lands on a coloured tile they will be given a corresponding card - either a Force Card (good) or a Trap Card (bad) & have to reap the reward or take the punishment.

Game End:

The game ends when with the first player reaching the Rebel Alliance base.
